# Nightbot Commands

> Replace `YOUR_WEB_APP_URL` below with your Apps Script web app URL.
> Use exactly this `$(urlfetch ...)` syntax in Nightbot messages.

**!queue**
```
$(urlfetch YOUR_WEB_APP_URL?queue=true)
```

**!wait**
```
$(urlfetch YOUR_WEB_APP_URL?queue=true)
```

**!spot**
```
$(urlfetch YOUR_WEB_APP_URL?spot=$(query))
```
